# MedievalRebellion · 中世纪农民起义

> 2D 横版动作 RPG：玩家从农场出发招募同伴，推翻地方官吏
> 迭代版本：V0.3.x（战斗实装 + 村庄背景 + 三角色编队 + 多波次通关）
> Godot 引擎：**4.6.2-stable_win64**

## ② 快速启动（3 个一键脚本，双击即可）

所有启动脚本位于 `cmd/v0.3/` 目录下，**首次使用请先编辑第 3 行 Godot 路径**：

```cmd
set "GODOT=D:\tools\game\Godot_v4.6.2\Godot_v4.6.2-stable_win64.exe"
```

> ⚠️ 若双击脚本出现「系统找不到指定的路径」或乱码：右键脚本 → 编辑 → 记事本「另存为」→ 编码选 **ANSI (GBK)** + CRLF 换行 → 覆盖保存。

| 脚本 | 用途 | 说明 |
|------|------|------|
| `cmd/v0.3/01-启动编辑器.cmd` | **启动 Godot 编辑器** | 调试所有场景；编辑器内按 `F5` 从主菜单运行游戏，`F6` 运行当前打开的场景 |
| `cmd/v0.3/02-启动游戏.cmd` | **直接运行游戏** | 无需打开编辑器，直接进入主菜单 → 选择战斗入口 |
| `cmd/v0.3/03-自动化验收V0.3.cmd` | **自动化测试** | 无头模式跑全量验收（V0.3i 共 6 Phase）；关闭 Godot 编辑器再执行，exit 0 = 全通过 |

### 命令行启动（可选）
```cmd
:: 打开编辑器
<Godot路径> --editor --path <项目根目录>

:: 直接启动游戏
<Godot路径> --path <项目根目录>
```

## ③ 玩家操作说明

### 战斗操作（默认按键映射）
| 操作 | 键盘按键 | 手柄 | 说明 |
|------|---------|------|------|
| 移动 | `A` / `D` 或 `←` / `→` | 左摇杆 | 左右地面移动（8 向框架） |
| 跳跃 | `W` / `↑` / `空格` | A 键 | 单段跳（可扩展二段跳） |
| 攻击 | `J` | X 键 | 3 段连击（第 3 段带击退） |
| 格挡 | `K` | B 键 | 正面 ±60° 减伤 80%，消耗体力；体力空则破盾双倍受伤 |
| 冲刺 | `L Shift` | LB 键 | 0.7s CD，20 体力，带无敌帧 |
| 切换武器 | `1`（耙子） / `2`（木剑） | RB 键循环 | 武器栏 2 选 1（见下方武器对比） |
| 拾取物品 | `E` | Y 键 | 靠近拾取，小金币/小血药自动拾取 |
| 同伴编队切换 | `Tab` / `3` `4` `5` | - | V0.3g 三角色切换：农民 / 樵夫 / 猎户 |
| 暂停 | `Esc` | Start 键 | 呼出菜单 + 暂停时间轴 |

### 武器对比（2 把可切换）
| 武器 | 伤害 | 攻速 CD | 体力消耗 | 说明 |
|------|------|---------|---------|------|
| 🧹 **耙子（默认）** | 13 - 16 | 0.32 秒 | 2 点 | 高伤慢打，第 3 段击退最强 |
| 🗡️ **木剑** | 5 - 8 | 0.18 秒 | 2 点 | 低伤快打，连段输出密度高 |

> 两种武器挥击体力消耗统一为 **2 点**。

### 伤害计算 7 步流水线（CDC 模块）
1. 原始伤害 × 连击倍率（第 1 段 100% · 第 2 段 110% · 第 3 段 130%）
2. 防御减伤：`max(0, 伤害 − 防御力 × 1.0)`
3. 格挡：正面按 K → 伤害 × 0.2（蓝字），体力扣空则破盾 ×2
4. 背刺：敌人背面 ±60° → 红字 **2×**（必暴击）
5. 暴击：默认 5% + 猎户 15% + 背面 20% → 黄字 **1.75×**
6. 钳制保底：下限 1 点 / 上限 = 目标最大 HP（防止秒人）
7. 击退向量：玩家被打 180 速 / 敌人被打 120 速 + 上扬 60px

## ④ 游戏内容

### 🏘️ 村庄背景（V01 场景默认）
向右无限动态生成的 RPG 风格村庄：
- **主路**：90px 宽土黄色土路 + 路肩 + 路面散石
- **建筑**：小屋别墅（烟囱冒烟）/ 两层小楼（街灯）/ 红色谷仓（草垛）/ 石头水井（吊桶）
- **农业**：农田（小麦 / 胡萝卜 / 玉米 3 种随机作物）+ 木栅栏（带村口告示牌）
- **远景**：远处绿色农田 + 村庄天际线（**中央教堂尖塔 + 白色十字架**）+ 太阳光晕 + 8 道光芒

### ⚔️ 敌人种类
| 敌人 | 外观 | 行为 | HP / ATK / DEF |
|------|------|------|----------------|
| 🌾 **稻草人兵** | 棕色布衣 + 短剑 | 巡逻 120px → 240px 发现追击 → 近战砍 | 60 / 7 / 1 |
| 🦅 **跳跃斥候** | 黄色轻甲 + 双匕首 | 巡逻 + 自动跳障碍 → 300px 追击 | 40 / 6 / 0 |
| 🏹 **弓箭手** | 稻草人同款 + 长弓 | 20 米内立刻响应：远程射箭 + 近距风筝后退 | 55 / 9 / 1 |
| 🛡️ **精英盾兵** | 蓝甲 + 大盾 | 正面格挡 80%，需绕后或破盾武器 | 120 / 11 / 5 |
| 🪵 **训练木桩** | 棕色矩形 | 站桩挨打，99999 HP，不掉落 | 99999 / 0 / 3 |

### 🤝 同伴系统（V0.3g 三角色编队）
| 同伴 | 定位 | 默认 AI 行为 |
|------|------|------------|
| 🧑‍🌾 **樵夫（斧兵）** | 近战肉盾 | 90px 跟随，260px 内冲锋近战；双手斧可强制破盾 |
| 🏹 **猎户（弓手）** | 远程输出 | 140px 跟随，320px 外射箭，保持 300px 最优射程 |
| 🐑 **牧人（辅助）** | 治疗辅助 | 60px 贴身跟随，每 5s 回 2% HP；HP<50% 给护盾 |

### 💎 掉落与金币
| 掉落物 | 颜色 | 自动拾取 | 效果 |
|--------|------|---------|------|
| 小金币 | #FFD166 亮黄 | ✅ | +1~3 金 |
| 中金币 | #F5A623 橙黄 | ✅ | +4~9 金 |
| 大金币 | #FF6B35 橙红 | ❌ 按 E | +10~20 金 |
| 小血药 | #EF476F 粉红 | ✅ | 回 25 HP |
| 中血药 | #D62828 深红 | ❌ 按 E | 回 60 HP |

> 高伤击杀（最后一击 ≥ 敌人 50% HP）→ 大金币掉落概率 +10%。

## ⑧ 开发说明

- 脚本语言：**GDScript**（Godot 原生，与 Python 语法接近）
- 渲染：2D 手绘几何（全部通过 `_draw()` 动态绘制，**零外部 sprite 依赖**，易改色改形）
- 代码风格：静态类型（`var x: int = 0`）+ 4 空格缩进，不使用 `const` 存可变值
- 分支策略：main 为主开发分支，需求完成自动本地 commit + push 到 origin/main
