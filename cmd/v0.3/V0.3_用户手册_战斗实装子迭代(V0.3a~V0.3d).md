## 二、V0.3d 键盘操作指南（6 组输入 → 11 状态可肉眼感知）

| 输入 | 功能 | 状态颜色 | 肉眼可见变化（UI可感知） |
| --- | --- | --- | --- |
| **A / D** | 左右移动 | 🟢 RUN 绿字 | 右上 STATE 蓝 IDLE → 绿 RUN；方向翻转 |
| **Space** | 跳跃（按2次=二段跳） | 🟠 JUMP 橙 / 🟣 DOUBLEJUMP 紫 | 1按→橙JUMP 上升；空中再按→紫DOUBLEJUMP 二次上升 |
| **J 键** | 攻击（3连击窗口=0.55s） | 🔴 ATTACK1/2/3 红 | 快速按J×3 → 依次切红ATTACK1→ATTACK2→ATTACK3；慢按>0.55s重置为ATTACK1 |
| **K 键**（长按） | 举盾格挡（每秒 -10 体力） | 🔵 BLOCK 蓝字 | 按下→蓝BLOCK；体力条每秒-10；松键→IDLE |
| **LeftShift** | 冲刺（CD=0.8s · 耗20体力 · 无敌帧） | 🔵 DASH 青字 | 按后→青DASH；体力条-20；0.8s内再按=CD拒 |
| **1 / 2 / 3** | 切武器：拳头/斧头/弓 | 武器颜色变化 | 1→灰拳头(×1.0) · 2→橙斧头(×1.3·破盾) · 3→绿长弓(×0.9·远程160) |
| **对面稻草人** | 每2秒自动反击玩家1次 | 若举盾=70%减伤 | 稻草人红拳→玩家HP减少；K举盾后减伤=仅-2 |

> ✅ **设计承诺**：全部 11 FSM 状态均有**独立颜色文字** + **HUD实时更新**，不需要查日志；事件日志区实时打印所有状态切换。

## 三、用户操作验证流程（6 步看出全部 UI 新功能）

### ✅ 步骤 1：测试 A/D 移动 → RUN 状态
操作：按住 D 键 1 秒，再按住 A 键 1 秒
验证：
- 右上 STATE 从 `IDLE 蓝` → `RUN 绿`
- 松开键回到 IDLE
- 日志区打印：`玩家FSM → RUN（肉眼可见颜色变化）`

### ✅ 步骤 2：测试 Space 双跳 → JUMP / DOUBLEJUMP
操作：站在地上 → 按 Space → 在空中再次按 Space
验证：
- 第 1 按 → STATE = `JUMP 橙` + 角色上升
- 第 2 按 → STATE = `DOUBLEJUMP 紫` + 角色二次上升
- 落地回到 IDLE

### ✅ 步骤 3：测试 J 键 3 连击 → ATTACK1/2/3
操作：快速连按 J 键 3 次（间隔<0.55秒）
验证：
- 依次显示：`ATTACK1 红` → `ATTACK2 红` → `ATTACK3 红`
- 等 0.7 秒后再按 J → 回到 ATTACK1（连击窗口超时重置）

### ✅ 步骤 4：测试 K 举盾 → BLOCK 蓝字
操作：按住 K 键 2 秒，再松开
验证：
- 按下瞬间 STATE = `BLOCK 蓝`
- 体力条 STA 每秒 -10（2 秒后从 100 → 约 80）
- 松键 → IDLE

### ✅ 步骤 5：测试 1/2/3 切武器 + Shift 冲刺
操作：先按 `1` → 再按 `2` → 再按 `3` → 最后按 LeftShift
验证：
- 武器标签依次变：灰拳头 → 橙斧头（破盾×1.3）→ 绿长弓（远程160）
- Shift 后 STATE = `DASH 青字`；体力 -20；0.8s 内再按=CD保护

### ✅ 步骤 6：攻击稻草人 + 盾破 + 反击
操作：切武器=2（斧头）→ 靠近稻草人 → 按 J 攻击
验证：
- 稻草人 HP 条减少，日志显示「稻草人HP → X / 150」
- 稻草人每 2 秒反击：若 K 举盾=仅 -2 HP（70%减伤）；不举盾=约 -6 HP
- 可按「🔄 重置全部」按钮满血复活双角色
