## 3. AI FSM 设计（Companion 4态）

### 3.1 同伴 AI 状态机（CompanionAIState enum，追加层）

```gdscript
enum CompanionAIState { FOLLOW_PLAYER, IDLE_NEAR, ASSIST_ATTACK, RETREAT }
```

| 状态 | 触发条件 | 行为 | 视觉反馈 |
|---|---|---|---|
| **FOLLOW_PLAYER** | `dist_player > follow_distance + 10` AND 无敌人 | `_move_toward(player)`，切 CharacterBase.RUN | 蓝🟦色块 |
| **IDLE_NEAR** | `dist_player <= follow_distance` AND 无敌人 | `velocity.x → 0`，CharacterBase.IDLE，恢复体力 | 青🟩色块 |
| **ASSIST_ATTACK** | 扫描到敌人（半径=alert_radius） | ① dist<=weapon.range → _do_attack → ATTACK1+挥弧动画<br>② dist>range → _move_toward(enemy) | 红🟥色块 |
| **RETREAT** | `dist_player > retreat_radius`（同伴跑太远） | 切 RUN 高速返回玩家，不追击 | 紫🟪色块 |

### 3.2 敌人扫描算法（无桩，纯递归遍历）
```
_find_nearest_enemy(origin, rng):
  SceneTree.root → 递归所有 CharacterBody2D
  过滤: node.kind == CharacterKind.ENEMY AND node != self AND node != follow_target
  取: 距离origin最近的 Node2D
```

### 3.3 攻击流程（接入 CDC7步）
```
_do_attack(enemy):
  ├─ if attack_cd_left > 0 → 退出
  ├─ change_state(ATTACK1)
  ├─ attack_cd_left = weapon.cd_sec（1.2秒斧手）
  ├─ opts = { _use_cdc:true, weapon_mult:1.2, break_shield:true, knockback:240 }
  └─ enemy.take_damage(atk=12, self, opts) → CDC 7步
```

## 5. 配置 companions.json 字段映射（V0.3e 樵夫取值）

| 字段 | 值 | 作用 |
|---|---|---|
| axeman.max_hp | 120 | Companion HP初始化上限 |
| axeman.base_atk | 12 | 传给 take_damage(atk=12) |
| ai.follow_distance | 90 | `dist<=90` → IDLE_NEAR |
| ai.alert_radius | 260 | 进入后切 ASSIST_ATTACK（扫描敌人） |
| ai.attack_range | 55 | dist<=55 → _do_attack |
| ai.retreat_radius | 340 | dist>340 → 强制RETREAT回玩家 |
| weapon.cd_sec | 1.2 | 挥斧冷却（避免一秒多砍） |
| weapon.atk_mult | 1.2 | CDC 第1步 `raw = atk × mult` |
| weapon.break_shield | true | 第③步格挡阶段绕过 |
