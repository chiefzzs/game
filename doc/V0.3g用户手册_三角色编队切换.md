## 二、操作清单（V0.3g Demo内）

### 基础（复用V0.3d）
| 按键 | 效果 |
|------|------|
| A / D 或 ← / → | 左右移动当前角色 |
| Space / W / ↑ | 跳 (按2次=二段跳DoubleJump) |
| Shift (长按) | 冲刺 Dash |
| J | 普攻 (农民=镰刀挥、锤兵=重锤圆、枪兵=直线刺) |
| K | 举盾格挡 (减伤70%) |

| 按键 | 效果 | 验证要点 |
|------|------|---------|

| **F1** | **选1号：农民（镰刀）** | 回到1号，HP卡[1]金光 |
| **F2** | **选2号：锤兵（钉锤高伤）** | HP 140（最厚），攻击范围圆 |
| **F3** | **选3号：枪兵（长枪脆皮）** | HP 90（最薄），攻击长条形 |

## 三、5步验收新功能（V0.3g 演示流程）

### Step1：进入V0.3g Demo
1. 双击 `cmd\v0.3\02-启动游戏.cmd` → 主菜单
2. 点最下面的 **第10橙按钮「👥 V0.3g 编队Tab切换」**
3. 你会看到：3个角色站成一排，左边农民脚下有大黄圈（=当前控制）

### Step2：用 F1/F2/F3 快速选角
- 按 **F1** → 农民被选中（黄圈1号）
- 按 **F2** → 锤兵被选中（黄圈2号，形象=灰甲+钉锤）
- 按 **F3** → 枪兵被选中（黄圈3号，红袍+长枪）
- ✔ 验证点：左上角3张HP卡，选中的那张边框变**金色**

### Step3：Tab 循环切人
- 从农民开始，按 **Tab**：黄圈→锤兵
- 再按 **Tab**：黄圈→枪兵
- 再按 **Tab**：黄圈→农民（回到起点）
- ✔ 验证点：底部金色大字 **当前角色名** 跟着变化

### Step4：每个角色独立控制 + 形象差异
| 操作 | 观察 |
|------|------|
| F1+J | 农民挥橙色镰刀弧线（长条） |
| F2+J | 锤兵举蓝色大钉锤（圆形范围） |
| F3+J | 枪兵红色直线刺（很长） |
| A/D | 只有当前选中的角色走动，其他两个站着 |

### Step5：3条HP独立
- 选锤兵F2，走到角落A/D。因为3条HP独立，锤兵HP：140（比农民的100厚）
- ✔ 左上角HP条：[1:100][2:140][3:90] 三条不同

## 四、常见问题（FAQ）

**Q1：主菜单找不到第10个按钮？**  
→ 滚动一下（如果屏小）。或确认V0.3g已经通过Headless测试=功能已合入。

**Q2：按Tab没反应？**  
→ 先点一下画面（确保窗口焦点）。只有Demo窗口激活时按键才有效。

**Q3：切到枪兵按J没看到长条形？**  
→ 枪兵攻击只有在**攻击动画进行中**才绘制长条形。快速按J+观察。

**Q4：V0.3f的史莱姆按钮还能玩吗？**  
→ **可以。** OneTrack保证：V0.3b~V0.3f全部9个按钮照常（已在提交前测过V0.3e=24/24、V0.3f=31/31）。
