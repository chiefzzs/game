## 一、项目概述

### 1.1 游戏类型
本项目为一款 **2D横向卷轴跳跃类动作游戏**，融合平台跳跃、即时战斗、队伍招募管理等多种玩法要素。

### 1.2 游戏主题
以 **欧洲中世纪** 为历史背景，玩家扮演一位革命者，从农场出发，招募同伴组建队伍，通过战斗推翻地方官吏的统治，最终控制地方政府。

### 1.3 核心玩法循环
1. **探索移动**：通过跳跃、冲刺等方式在平台关卡中前进
2. **招募同伴**：通过对话或金钱方式招募NPC加入队伍
3. **战斗对抗**：与敌人进行即时战斗，利用武器和技能击败对手
4. **推进目标**：逐步接近并击败地方官吏，控制地方政府

## 二、背景故事与世界观

### 2.1 时代背景
- **时间设定**：欧洲中世纪（约公元11-15世纪）
- **社会背景**：封建领主割据，地方官吏横征暴敛，民不聊生
- **核心矛盾**：底层民众与腐朽统治阶层之间的阶级矛盾

### 2.2 游戏起点
- **出生地点**：一个宁静的乡村农场
- **初始状态**：玩家是农场中的普通农民/劳动者
- **触发事件**：目睹地方官吏的暴行，决心反抗暴政

### 2.3 游戏终极目标
- **主线目标**：击败当地最高地方官吏（如：领主、男爵、郡守等）
- **达成结果**：控制地方政府，建立新的秩序

## 三、操作系统

### 3.1 控制器按键映射（以Xbox手柄为基准）

| 按键/摇杆 | 功能 | 说明 |
|-----------|------|------|
| **L 摇杆**（左摇杆） | 移动 | 控制角色左右水平移动 |
| **A 键** | 跳跃 | 短按=小跳，长按=高跳 |
| **Y 键** | 冲刺 | 快速向当前方向冲刺一段距离，可用于躲避或接近敌人 |
| **X 键** | 攻击 | 使用武器进行普通攻击 |
| **LT 键**（左扳机） | 格挡 | 按住=进入格挡防御姿态，松开=解除格挡 |

### 3.2 操作机制详解

#### 3.2.1 移动系统
- **水平移动**：通过左摇杆控制角色左右移动，具备加速度和减速度
- **移动速度**：基础移动速度、冲刺速度两档
- **方向感知**：角色朝向跟随摇杆方向或攻击方向

#### 3.2.2 跳跃系统
- **基础跳跃**：按A键触发单段跳跃
- **跳跃高度**：按键时长影响跳跃高度（短按小跳，长按高跳）
- **空中控制**：跳跃中可通过摇杆微调水平位置
- **落地判定**：具备落地缓冲动画

#### 3.2.3 格挡系统
- **触发方式**：按住LT键（左扳机）进入格挡姿态
- **格挡效果**：抵消正面攻击伤害
- **格挡限制**：
  - 格挡时无法移动
  - 格挡时无法攻击
  - 持续格挡消耗耐力值（可选设计）
  - 背面攻击无法格挡

#### 3.2.4 冲刺系统
- **触发方式**：按Y键触发
- **冲刺效果**：
  - 向当前移动方向快速位移
  - 冲刺过程中具有短暂无敌帧（躲避伤害）
  - 可在空中使用
- **冷却机制**：冲刺后有短暂冷却时间（约1-2秒）

#### 3.2.5 攻击系统
- **基础攻击**：按X键挥砍武器
- **攻击连击**：连续按X键可触发连击（最多3段）
- **攻击判定**：攻击具有前方扇形/矩形判定区域
- **攻击硬直**：攻击动作有前后摇时间

## 四、角色系统

### 4.1 玩家角色

#### 4.1.1 基础属性
| 属性 | 说明 |
|------|------|
| **生命值（HP）** | 角色生命值，归零则角色死亡/游戏失败 |
| **攻击力（ATK）** | 决定普通攻击的伤害数值 |
| **防御力（DEF）** | 减少受到的伤害（可选） |
| **移动速度** | 基础移动速度 |
| **跳跃力** | 决定跳跃高度 |

#### 4.1.2 死亡与失败条件
- 玩家HP归零 → 游戏失败，回到检查点/重新开始

### 4.2 可招募同伴系统

#### 4.2.1 招募机制
- **招募方式**：
  1. **对话招募**：与特定NPC对话，完成对话选项或小型任务后加入
  2. **金钱收买**：支付一定金币直接招募
- **招募前提**：部分角色需要满足特定条件（如：声望、任务进度）

#### 4.2.2 同伴属性体系
| 属性 | 说明 | 与价格关系 |
|------|------|------------|
| **收购价格** | 招募该角色所需金币数量 | 血量越多，价格越高 |
| **生命值（HP）** | 同伴的血量上限 | 核心属性，决定价格的主要因素 |
| **攻击力（ATK）** | 同伴的攻击伤害 | 可随血量/价格浮动 |
| **攻击类型** | 近战/远程/辅助 | 不同类型有不同战斗行为 |
| **移动速度** | 同伴跟随移动的速度 | - |

#### 4.2.3 同伴行为规则
- **跟随行为**：自动跟随玩家移动，保持一定距离
- **战斗AI**：
  - 玩家攻击敌人时，同伴自动协助攻击
  - 敌人靠近时，同伴主动迎击
  - 优先攻击与玩家交战的目标
- **生命机制**：
  - ❌ **不会自动恢复血量**
  - ❌ 无治疗道具恢复（根据需求）
  - ✅ 血量清零时，同伴 **永久消失**（死亡/离队）
  - ✅ 消失后无法再次招募同一角色

#### 4.2.4 同伴分级示例（参考）
| 等级 | HP范围 | 价格区间 | 特点 |
|------|--------|----------|------|
| 普通 | 30-50 | 20-50金币 | 低血量、低攻击、价格便宜 |
| 精英 | 60-100 | 80-150金币 | 中等血量、有特殊技能 |
| 稀有 | 120-200 | 200-400金币 | 高血量、高攻击、特殊能力 |

## 五、战斗系统

### 5.1 战斗体系定位
- **战斗模式**：即时动作战斗（非回合制）
- **战斗场景**：在探索过程中遭遇敌人即进入战斗（无缝切换）
- **战斗场地**：利用平台地形进行立体战斗

### 5.2 战斗核心机制

#### 5.2.1 伤害计算
```
实际伤害 = 攻击力 - 敌人防御力（若有）- 格挡减伤
```

#### 5.2.2 战斗状态
| 状态 | 说明 |
|------|------|
| **普通状态** | 正常移动、攻击、跳跃 |
| **硬直状态** | 受击后短暂无法操作 |
| **格挡状态** | 按住LT键，正面减伤 |
| **冲刺状态** | Y键触发，短暂无敌 |
| **死亡状态** | HP归零，角色倒地 |

#### 5.2.3 武器系统
- **初始武器**：简易农具/短剑
- **武器获取**：可通过购买、掉落、宝箱等方式获得更强武器
- **武器属性**：攻击力、攻击速度、攻击范围

### 5.3 敌人系统

#### 5.3.1 敌人类型
| 类型 | 特点 | 出现场景 |
|------|------|----------|
| **普通士兵** | 低血量、低攻击、成群出现 | 道路、关卡入口 |
| **精英守卫** | 高血量、高攻击、有护甲 | 重要建筑附近 |
| **骑士/重甲兵** | 极高防御、需要破防 | 城堡区域 |
| **弓箭手** | 远程攻击、血量低 | 高处、城楼 |
| **地方官吏（BOSS）** | 最终BOSS，多阶段战斗 | 政府/城堡最深处 |

#### 5.3.2 敌人AI行为
- **巡逻**：在固定区域来回巡逻
- **警戒**：玩家进入视野后，警觉并准备战斗
- **追击**：发现玩家后追击攻击
- **攻击**：进入攻击范围后发动攻击
- **撤退**：部分低血量敌人可能逃跑

## 七、关卡与地图设计

### 7.1 关卡结构（建议）
1. **农场区域**（新手教学区）
   - 熟悉操作按键
   - 基础跳跃练习
   - 招募第一个同伴

2. **村庄/城镇边缘**
   - 遭遇第一批敌人
   - 招募更多同伴
   - 收集金币

3. **城镇区域**
   - 商店系统
   - 更多NPC角色
   - 中等难度敌人

4. **行政区/城堡外围**
   - 精英敌人
   - 复杂平台跳跃
   - 重要同伴招募

5. **地方政府/城堡**（最终关卡）
   - BOSS战：地方官吏
   - 游戏结局

### 7.2 平台跳跃要素
- 普通平台
- 移动平台
- 可破坏平台
- 高处/低处落差
- 跳跃间隙（需要冲刺跳跃）

## 八、游戏流程

### 8.1 完整流程概览
```
开始游戏
  ↓
【序章：农场】
  ├─ 过场动画/文字介绍背景
  ├─ 操作教学（移动、跳跃、格挡、冲刺、攻击）
  ├─ 与第一个NPC对话
  └─ 完成第一个简单任务
  ↓
【第一章：启程】
  ├─ 离开农场，进入野外
  ├─ 首次战斗（基础敌人）
  ├─ 学习招募系统
  └─ 招募1-2名同伴
  ↓
【第二章：发展】
  ├─ 探索村庄/城镇
  ├─ 积累金币
  ├─ 招募更多同伴（3-5人）
  ├─ 购买/升级武器
  └─ 多次战斗锻炼
  ↓
【第三章：渗透】
  ├─ 接近行政区
  ├─ 应对精英敌人
  ├─ 同伴可能牺牲（永久消失）
  └─ 战略性招募补充
  ↓
【最终章：决战】
  ├─ 攻入地方政府/城堡
  ├─ 连续战斗
  ├─ BOSS战：地方官吏
  └─ 击败BOSS → 控制地方政府
  ↓
游戏通关（胜利结局）
```

### 8.2 新手教学流程
1. 玩家在农场醒来
2. 系统提示："使用左摇杆移动" → 练习移动
3. 系统提示："按A键跳跃" → 跳过栅栏/障碍
4. 系统提示："按住LT键格挡" → 格挡飞来的物体
5. 系统提示："按Y键冲刺" → 冲刺穿过狭窄通道
6. 系统提示："按X键攻击" → 攻击稻草人/木桩
7. 教学完成，开始正式游戏

## 十、非功能性需求

### 10.1 性能需求
- 稳定帧率：60FPS
- 加载时间：单场景加载 < 3秒
- 内存占用：合理控制

### 10.2 操作体验
- 输入延迟 < 50ms
- 打击反馈：受击特效、屏幕震动
- 跳跃手感：精确的起跳和落地判定

### 10.3 存档系统（建议）
- 自动存档：通过检查点时自动保存
- 手动存档：暂停菜单可保存
- 存档内容：玩家位置、HP、金币、队伍成员、进度
