## 一、角色详细设计（17 名）

### 1.1 主角（玩家角色）

| 属性 | 值 | 技术对应（CharacterBase） |
|---|---|---|
| 名称 | **布衣农夫（默认名 John，可改名）** | `kind = "player"`（不在编辑器 6 类实体里，玩家为单例） |
| 视觉参考 | 马里奥体态（圆润 144×144）：土黄布衣 + 棕色草帽 + 光脚/草鞋；手持武器初始=农用钉耙 → 1-1 训练后换木质长剑 → 3-1 破城门换铁剑 | `CharacterBase.sprite`，动画：idle/run/jump/attack_1/attack_2/hurt/die |
| 初始数值（`L2_balance/player.json`） | `max_hp=100 / stamina=100 / atk=8 / def=2 / speed=260 / jump_force=-520 / dash_force=420` | CharacterBase 初始化参数 |
| 核心技能 | ① 二段跳（长按跳更高） ② 冲刺（消耗 20 体力，CD 0.8s） ③ 格挡（LT 按住，减伤 80%，体力每秒 -15） ④ 三连击（X×3，第三段击退 1.5×） | InputBus 对应 `jumped / dashed / block_changed / attacked` 信号 |

### 1.2 可招募队友（3 名，1-1 关集齐，4-1 陆续离场）

> **机制**：3 名队友非玩家直接控制，AI 跟随（距离≤3 格），自动攻击范围内敌人，HP 独立，共享 `Saves` 存档。

| 编号 | 身份 | 编辑器实体模板（kind=`npc`） | 武器 / 战斗风格 | 数值 | 离场剧情 |
|---|---|---|---|---|---|
| C1 | **樵夫（Axeman）** | `npc_axeman`（继承自 `npc_farmer`，props 改 `atk=12 hp=120 icon_color=Color(0.5,0.3,0.1)`） | 双手斧：慢速重砍，击退 2×，破盾（3-1 破重甲盾兵必备） | HP 120 / ATK 12 / 攻速 1.2s | 4-1 鎏金长廊第二段，为了挡掉落石被压退场（画面模糊 + 声音低沉） |
| C2 | **猎户（Hunter）** | `npc_hunter`（props `atk=9 hp=70 patrol_r=0 ranged=true`） | 短弓：远程，射速快，单体伤害低，攻击飞行炮弹类敌人（3-1 弓箭手对射） | HP 70 / ATK 9 / 弹速 620 | 4-1 鎏金长廊第三段，被两名精英骑士合力一击击退掉进深坑 |
| C3 | **牧人（Shepherd）** | `npc_shepherd`（props `atk=6 hp=80 heal_per_sec=2 support=true`） | 牧羊杖：近战；每 5s 对全队回复 2% HP；格挡时可替主角承受 50% 伤害 | HP 80 / ATK 6 / HPS 2 | 4-BOSS 开场时，替主角挡领主第一招重锤，飞出屏幕外，只剩主角一人（对应故事情节「仅剩主角一人进场」） |

> **技术落地**：3 个 NPC 新建到 V0.3 的 `scripts/characters/CompanionBase.gd`，继承 CharacterBase，AI 用简单 FSM（跟随→警戒→攻击→回防），不做复杂行为树。

## 三、地图场景详细设计（7 关 + 1 隐藏关）

> **通用 8 层 TileMap 语义（所有关一致，对齐 V0.2 WorkshopMain 中 LAYER_NAMES 定义）**：
> - L0 天空背景：渐变 / 远山 / 白云（纯 ColorRect 或纯色瓦片，**无碰撞**）
> - L1 远景装饰：远处建筑轮廓 / 树冠 / 森林远景（**无碰撞**，视差滚动 0.4× 速度）
> - L2 地面：草地 / 泥土 / 石路 / 城砖（**视觉，不单独加碰撞**，碰撞统一用 MapLoader 生成的地板 StaticBody2D）
> - L3 近景装饰：花 / 杂草 / 碎石 / 萤火虫（**无碰撞**）
> - L4 建筑墙体：农舍墙壁 / 树干 / 城墙石 / 柱子（**有碰撞，SOLID_WALL_TILE_IDS**）
> - L5 建筑屋顶 / 树冠 / 城垛：茅草顶 / 红屋顶 / 藤蔓 / 城垛（**部分有碰撞 = 浮空平台，玩家可跳上去**）
> - L6 交互物 / 特效层：告示牌 / 训练场靶 / 锁块 / 荷叶动画（**部分有碰撞=实体，或 trigger 专用**）
> - L7 前景遮罩：树叶遮挡 / 烛台前景 / 门帘（**无碰撞，z-index > 玩家**）
